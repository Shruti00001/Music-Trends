Music Trends Analytics

A data-driven analysis of modern music trends and forecasting.

August 2024 Python, SQL, Tableau, Power BI

Project Overview

This project explores music trends by analyzing datasets from Spotify, YouTube, and TikTok. Using Python and SQL, I built a data pipeline to clean and structure the data. I then developed Tableau and Power BI dashboards to visualize evolving genre popularity and key artist trends. The project also utilized machine learning models to forecast future music engagement.
